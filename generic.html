Projects

Yike

Yike is a e-commerce shoe clone site of Nike with a Kobe theme. This was my first full stack project using a Ruby on Rails backend and a React/Redux frontend. The backend is connected to Amazon Web Services S3 to optimize storage and maintain security through AWS IAM. Test out the site and add shoes to your cart through the live link!

Live | Git

WeHaul

WeHaul is a Uber-type web application where users can sign up as a user or hauler. As a user, one can request a driver to pick them up and rate him or her after the job is completed. As a driver, he or she can view the list of jobs and accept a request. This MERN app uses a MongoDB and Express.js backend with a React/Redux frontend. Go experience Wehaul!

Live | Git

Hokage

Hokage is a Javascript infinite run game where you will venture through a forest and avoid obstacles while eliminating enemies on your journey to become Hokage. This game is fully built on Javascript and HTML5 Canvas. A Parallax background is used to simulate constant scrolling and animation of 2D sprites. Embark on your journey with the live link. Believe it!
